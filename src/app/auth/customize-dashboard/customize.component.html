Customize Your Dashboard

Here you can customize your dashboard. Please keep the fields selected which you will like to view in Issues List.

Also note that, you can always view the full Issue details by clicking on the Issue number irrespective of your customized field selection in dashboard.

  • Issue Description
  • Issue Severity
  • Issue Status
  • Issue Created Date
  • Issue Resolved Date
  • Issue Created By
  • Issue Last Modified By
